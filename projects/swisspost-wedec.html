SwissPost - Digital Commerce API

2013 â€“ 2014

Why

SwissPost wants to provide digital addressing and logistic services in a nutshell to the e-commerce market segment.

What

To design and develop a public web API for integration into e-commerce websites of Switzerland.

This B2B2C API includes the Swiss Post Login as an authentication service and provides a secure exposition of the addressing and logistic services of the Swiss Post.

Requirements

  • To define a public API to expose end user's personal delivery addresses defined in his SwissPost identity.
  • To define a public API to expose delivery options by the SwissPost.
  • To restrict the access to the public API to both marchant and end user.
  • To ask the consent of the end user before returning his personal data.
  • To integrate the API with the Swiss Post Login connector component as identity provider.

Who, where

The Digital Commerce API should be integrated by e-commerce websites of Switzerland.

Organization

Team of 3 software engineers, 1 project manager, 1 software architect (me).
